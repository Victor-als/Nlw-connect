## Tenologias usadas no projeto

- Next.js (React.js)
- Tailwind
- React hook form
- postgreSql
- Zod


## Para rodar o projeto localmente

1. Clone o repositório:

```sh
$ git clone https://github.com/Victor-als/Nlw-connect
```

3. Instale as dependências:

```sh
npm install
```

4. Rode localmente:

```sh
npm run dev
```
